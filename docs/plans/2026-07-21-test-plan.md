## 2. Altyapı Kurulumu

### 2.1 pytest ve pytest-cov Kurulumu
- `requirements.txt`'den `pytest>=8.0.0` ve `pytest-cov>=5.0.0` uncomment et
- `.venv/bin/pip install pytest pytest-cov`

### 2.2 pyproject.toml Konfigürasyonu
```toml
[tool.pytest.ini_options]
testpaths = ["tests"]
python_files = ["test_*.py"]
python_classes = ["Test*"]
python_functions = ["test_*"]
addopts = "-v --tb=short --strict-markers"
markers = [
    "unit: Unit tests (no external dependencies)",
    "integration: Integration tests (may use temp DB)",
    "slow: Tests that take >1s",
]

[tool.coverage.run]
source = ["src"]
omit = ["tests/*", "src/web/*"]

[tool.coverage.report]
show_missing = true
fail_under = 70
```

## 4. Test Dosyaları — Detaylı Plan

### Öncelik 1: Pure Logic (Mock yok, hızlı)

#### 4.1 `test_schemas.py` — Pydantic Schema Testleri
- `test_position_frozen`: Position instance immutable
- `test_position_required_fields`: Zorunlu alanlar olmadan hata
- `test_config_defaults`: Config default değerleri doğru
- `test_trade_history_entry`: TradeHistoryEntry validasyon

#### 4.2 `test_constants.py` — Constants Testleri
- `test_constants_exist`: Tüm kritik sabitler tanımlı
- `test_fib_values`: Fibonacci değerleri doğru (3, 5, 8, 13, 21, 34, 55)
- `test_thresholds_in_range`: Threshold değerleri mantıklı aralıkta

#### 4.3 `test_indicators.py` — Teknik İndikatör Testleri
- `test_calculate_rsi_series`: RSI serisi boyutu ve aralığı (0-100)
- `test_calculate_ema_series`: EMA serisi boyutu
- `test_calculate_adx`: ADX pozitif ve mantıklı aralıkta
- `test_calculate_atr_series`: ATR serisi pozitif
- `test_calculate_bollinger_bands`: BB upper > middle > lower
- `test_calculate_obv`: OBV trend yönü doğru
- `test_calculate_supertrend`: Supertrend yönü BULLISH/BEARISH
- `test_calculate_vwap`: VWAP pozitif
- `test_calculate_efficiency_ratio`: ER 0-1 aralığında
- `test_generate_smart_sparkline`: Sparkline yapısı doğru

#### 4.4 `test_regime_detector.py` — Regime Sınıflandırma
- `test_classify_bullish`: price > ema20 + ADX > 25 → BULLISH
- `test_classify_bearish`: price < ema20 + ADX > 25 → BEARISH
- `test_classify_neutral`: ADX < 25 → NEUTRAL
- `test_classify_choppy`: ER < 0.30 → CHOPPY
- `test_overall_regime`: Çoğunluk BULLISH ise → BULLISH
- `test_regime_strength`: Alignment hesaplama doğru

#### 4.5 `test_utils.py` — Yardımcı Fonksiyonlar
- `test_safe_file_read`: Var olan dosyayı okur
- `test_safe_file_read_missing`: Olmayan dosyada default döner
- `test_safe_file_write`: Dosyaya yazar ve okunabilir
- `test_format_num`: Sayı formatlaması doğru
- `test_retry_manager`: Retry mekanizması çalışır
